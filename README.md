# crc16

crc16 for Crystal

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  crc16:
    github: maiha/crc16.cr
```

## Usage


```crystal
require "crc16"

Crc16.crc16("foo")  # => 44950
```
